## EVE Timerboard
### Overview
The `django_eveonline_timerboard` allows you to create and track timers for EVE events. 

### Quick Steps 
1. Install `django_eveonline_timerboard`
2. No additional setup required

### Permissions
Users will need permissions to view timers.

|    Permission   |   Action    |
|  ---  |  ---  |
| Can view eve timer  |   Ability to view timers  |
| Can add eve timer   |  Ability to add timers   |
| Can delete eve timer   |  Ability to delete timers   |
